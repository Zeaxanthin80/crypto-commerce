Project Overview

This cryptocurrency e-commerce platform allows users to browse products, add them to cart, and make purchases using USDT cryptocurrency.

The project is built with React.js and Next.js for the frontend, using Tailwind CSS for styling. It includes user authentication, product management, shopping cart functionality, and cryptocurrency payment processing.

Below you'll find demonstrations of the key pages that have been implemented.

Next Steps

To properly set up and run the project:

  1. Resolve the PowerShell execution policy issue by opening PowerShell as Administrator and running: 
    Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
  2. Install Node.js dependencies from the command line: 
    cd C:\Users\Jose\CascadeProjects\crypto-commerce\frontend
npm install
  3. Start the development server: 
    npm run dev
  4. Open your browser and navigate to http://localhost:3000
